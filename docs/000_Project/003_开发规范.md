# 003_开发规范

## 文档信息

-   项目：YDSmart 印染智能工厂平台
-   版本：V1.0
-   日期：2026-07-03

# 1. 开发目标

建立统一的开发规范，保证代码一致性、可维护性和可扩展性。

# 2. 技术栈

## 前端

-   Vue3
-   TypeScript
-   Vite
-   Pinia
-   Ant Design Vue
-   Axios

## 后端

-   ASP.NET Core
-   Entity Framework Core
-   AutoMapper
-   FluentValidation
-   Serilog
-   Swagger

# 3. 架构原则

-   Clean Architecture
-   DDD（适度）
-   SOLID
-   Repository Pattern
-   Unit of Work
-   依赖注入

# 4. 编码规范

-   开启 Nullable
-   开启 XML 注释
-   一个类一个职责
-   禁止重复代码
-   禁止硬编码
-   禁止字符串拼接 SQL
-   所有配置放入 appsettings

# 5. API 规范

-   RESTful
-   JSON UTF-8
-   统一返回 Result`<T>`{=html}
-   使用 HTTP 状态码
-   必须提供 Swagger

# 6. 数据库规范

-   SQL Server
-   主键 BIGINT
-   审计字段：CreateTime、CreateUser、UpdateTime、UpdateUser、IsDeleted
-   建立必要索引
-   使用 EF Core Migration

# 7. 日志规范

-   Info
-   Warning
-   Error
-   Audit
-   Exception

# 8. 测试规范

-   Service 层必须有单元测试
-   关键业务需要集成测试

# 9. Git 规范

-   main：生产
-   develop：开发
-   feature/\*
-   hotfix/\*
-   release/\*

提交信息： - feat: - fix: - refactor: - docs: - test:

# 10. Cursor 开发要求

Cursor 生成代码必须： - 保持分层 - 自动生成
Entity、DTO、Repository、Service、Controller - 添加中文注释 -
保证可编译 - 不生成伪代码
