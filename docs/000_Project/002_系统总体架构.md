# 002_系统总体架构

> 项目：YDSmart 印染智能工厂平台

## 1. 架构目标

构建 ERP + APS + MES + EMS + BI + AI
一体化平台，采用前后端分离架构，支持高并发、可扩展、可维护。

## 2. 总体架构

``` text
Vue3 + TypeScript
        │
 ASP.NET Core Web API
        │
 Application Layer
        │
 Domain Layer
        │
 Infrastructure Layer
        │
 SQL Server / Redis / RabbitMQ
        │
 PLC / IoT / 第三方ERP
```

## 4. 分层设计

### Presentation

Web UI、移动端、开放 API。

### Application

业务用例、DTO、服务编排。

### Domain

聚合、实体、值对象、领域服务、业务规则。

### Infrastructure

EF Core、缓存、消息、文件、第三方接口。

## 6. 通信方式

-   REST API

## 7. 安全设计

-   JWT 身份认证

-   操作审计
-   数据权限

## 8. 部署建议

开发：Docker Compose

生产： - Nginx - ASP.NET Core - SQL Server - Redis - RabbitMQ - MinIO

## 9. Cursor 开发要求

-   严格遵循 Clean Architecture。
-   所有业务必须通过 Application 层。
-   禁止跨层直接访问数据库。
-   API 返回统一 Result`<T>`{=html}。
-   每个模块包含 Entity、DTO、Repository、Service、Controller、Unit
    Test。
